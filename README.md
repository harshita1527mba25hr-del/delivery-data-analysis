Delivery Data Analysis
##  Overview
This repository contains the coursework for **Week 2 Lab 21-22**, focusing on analyzing delivery performance data.  
The project explores how traffic levels, vehicle types, and other factors influence delivery efficiency, using Python-based data analysis and visualization.

---

##  Objectives
- Inspect and summarize the delivery dataset  
- Calculate average delivery times under different traffic conditions  
- Compare fuel costs across vehicle types  
- Visualize delivery trends with clear plots and charts  
- Apply analytical thinking to real-world logistics scenarios  

---

##  Tools & Libraries Used
- **Python 3**  
- **Pandas** – for data handling and grouped statistical analysis  
- **Matplotlib** – for visualizations (bar charts, plots)
